Tools are offered per turn by intent route (see
[Architecture](/concepts/architecture)). **Ask first** = pauses on an in-chat
Approve/Deny card by default; all defaults are overridable per tool under
**Brain → Tools**.

## Knowledge & memory

| Tool | Does | Ask first |
|---|---|---|
| `search_graph` | semantic search over notes, screen activity, memories, and past conversations | — |
| `understory_search` | search published articles in The Understory | — |
| `understory_today` | latest edition's headlines | — |
| `now` | current date/time | — |
| `remember` | save a typed memory (preference/fact/goal/person/routine) | — |
| `recall` | list/filter saved memories | — |

## Computer use

Offered only on computer-intent turns; file tools are jailed under
`LEASH_COMPUTER_ROOT` (default: home).

| Tool | Does | Ask first |
|---|---|---|
| `screenshot` | capture screen → local VLM describes it (frame deleted) | — |
| `read_file` | read a text file (64 KB cap, jailed) | — |
| `write_file` | create/replace a text file (1 MB cap, jailed) | ✅ |
| `edit_file` | exact-string replacement (unique-match required, jailed) | ✅ |
| `run_command` | `bash -c` (60 s kill, 16 KB output cap, stripped env) | ✅ |
| `computer` | mouse/keyboard via cliclick (move/click/type/key/scroll) | ✅ |

## MCP

Tools from connected MCP servers (Brain → MCP) merge into the registry under their
server's namespace. The bundled mesh server contributes LAN-pairing tools whose PIN
entry renders as an in-chat form.
