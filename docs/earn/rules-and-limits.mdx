---
title: "Rules and limits"
description: "The boundaries that keep private mesh delegation free and public mesh earning controlled."
---

The economy is intentionally bounded. It should make public compute useful without turning private delegation into a billing problem.

## Mesh limits

| Limit | Value |
|---|---:|
| Private meshes per user | `1` |
| Public meshes per user | `15` |
| Total mesh slots | `16` |

The private mesh is the personal trust boundary. Public meshes are market boundaries.

## Charging rules

- Local device inference is always free.
- Private mesh inference is always zero-rate.
- Public mesh inference can be free or paid.
- A public provider with rate `0` does not require payment authorization.
- A public provider with a non-zero rate requires pre-authorization before streaming.
- Paid settlement is based on metered usage, not just the maximum cap.
- The current `/economy` ledger and replicated receipts are paid-session-first.

## Safety gates

The router can only move from private to public when the request passes its sensitivity and confidence filters.

Public routing should be blocked when:

- The prompt includes private files, credentials, secrets, or personal records.
- The task requires local state mutation or privileged tools.
- The classifier cannot confidently mark the turn public-safe.
- The requested model or modality is not advertised by the provider.
- The consumer's spend cap would be exceeded.

## Spend controls

The runtime-enforced spend controls today are:

- `HYPHA_ECONOMY_FLOAT` — reservation ceiling per session.
- `HYPHA_ECONOMY_MAX_PER_TX` — max per settlement.
- `HYPHA_ECONOMY_MAX_PER_HOUR` — hourly ceiling.
- `HYPHA_ECONOMY_MAX_PER_COUNTERPARTY` — per-provider ceiling.
- Optional `computeBudget` on a request when a caller explicitly passes one.

## Provider controls

Providers should be able to choose how they participate:

- Join no public meshes and serve only private capacity.
- Join a public mesh with a `0` rate.
- Join a public mesh with a paid rate.
- Advertise only the aliases the local serve exposes.
- Stop or pause Hypha / QVAC when the device should not serve.

## Current implementation note

The runtime already models private zero-rate routes and public free-or-paid routes in the economy
policy. The paid public-session path, settlement retries, and ledger are real. Zero-rate public
routes exist, but the public economy UI is still paid-session-first, so use the
[Economy platform guide](/platforms/economy) and smoke tests when validating provider behavior.
