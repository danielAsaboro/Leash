## Core claim set

These are the claims the pushed repo can support on its own:

- Leash is a **general-purpose personal AI harness** over the **Mycelium** runtime.
- All AI paths run **through `@qvac/sdk` only**; there is **no cloud AI** path.
- The assistant **perceives local context** through notes, voice, OCR/photos, screenshots, recent
  activity, and its feed.
- It **reasons above one device's weight class** by borrowing compute across an encrypted mesh.
- On your **private mesh**, your own devices delegate for free. On a **public mesh**, outside
  providers can expose compute free or paid.
- After warm-cache, the hot path is **offline-capable**.

## Evidence files worth opening first

Start with the structured bundle. It is organized by capability and device:

```text
evidence/
  manifest.json
  chats/{mac-mini,mbp}/
  qvac/{mac-mini,mbp}/
  hypha/{mac-mini,mbp}/
  leash/{mac-mini,mbp}/
  data/{mac-mini,mbp}/
```

Open these first:

| File                                   | Why it matters                                                         |
| -------------------------------------- | ---------------------------------------------------------------------- |
| `evidence/manifest.json`               | Top-level counts, generated files, date windows, and redaction summary |
| `evidence/qvac/mac-mini/summary.json`  | QVAC model loads, completions, embeddings, RAG, delegation, LoRA       |
| `evidence/qvac/mbp/summary.json`       | Same primitive proof set from the MacBook Pro side                     |
| `evidence/hypha/mac-mini/summary.json` | Mesh pairing, delegation, completion, and adapter events               |
| `evidence/hypha/mbp/summary.json`      | MacBook Pro mesh-side event summary                                    |
| `evidence/chats/mac-mini/index.json`   | Chat count, message count, date range, and roles                       |
| `evidence/data/mbp/summary.json`       | Folded MBP runtime/data evidence with redacted credentials             |

Other supporting files:

- `spike/logs/*.jsonl` for the original primitive proof runs.
- `evidence/medpsy-demo.jsonl` for a committed end-to-end health-record RAG example.
- `evidence/remote-api-calls.json` for machine-readable outbound-call disclosure.
- [Benchmarks](/reference/benchmarks) for performance snapshots in docs form.
