# Project Mycelium — facts

Mycelium is a private, offline, end-to-end-encrypted exocortex across a device
mesh. It is built entirely on the QVAC SDK and submitted to QVAC Hackathon I.

- The closed loop is Senses → Mind → Memory → sharper Senses.
- The "council" is 2–4 small models with the roles proposer, critic, specialist,
  and verifier, debating with tool-calling over a RAG-grounded context graph.
- The demo's mic-drop moment is putting every device in airplane mode and showing
  it keeps sensing, reasoning, and learning.
- The project license is Apache-2.0.
